# E-Commerce Store Front-end

The E-Commerce Store Front-end is a modern and user-friendly platform that provides a seamless shopping experience to customers. It is built on top of the powerful e-commerce dashboard and CMS, allowing users to browse and purchase products with ease.

## Build with

- Next.js
- TypeScript
- Prisma
- MySQL
- Clerk
- Shadcn UI
- Tailwind CSS

## Key Features

- **Shadcn UI for the Store**: The store front-end features a beautiful and responsive design powered by Shadcn UI.
- **Integrates with the E-Commerce Dashboard**: The front-end connects seamlessly with the [E-Commerce Dashboard & CMS](https://ecommerce-store-cms.vercel.app/), ensuring accurate product information and inventory management.
- **Multi-Vendor Support**: Customers can browse and purchase products from multiple vendors or stores managed through the CMS.
- **Category and Product Management**: Customers can explore various categories and view detailed product information, including images, descriptions, and pricing.
- **Flexible Image Support**: Vendors can upload multiple images for products and update them at any time.
- **Filtering and Searching**: Customers can easily search for products by categories, sizes, colors, and keywords, with pagination support for efficient browsing.
- **Featured Products**: The front-end highlights featured products on the homepage to attract customer attention.
- **Order Placement and Management**: Customers can create orders and track their status through an intuitive interface.
- **Secure Checkout with Stripe**: Stripe integration enables smooth and secure payment processing during the checkout process.
- **Revenue Analytics**: Graphs and statistics provide vendors with insights into their sales and revenue performance.
- **Authentication**: Clerk Authentication ensures secure access to customer accounts and order history.
- **Deployment**: The project is deployed on Vercel, providing a live preview at [ecommerce-store-cms.vercel.app](https://ecommerce-store-cms.vercel.app/).
- **ADMIN URL**:  The Admin project link is at [dashboard-cms](https://devdzportfolio.vercel.app/projects/dashboard-cms).


## Conclusion

The E-Commerce Store Front-end provides a user-friendly and feature-rich interface for customers to browse and purchase products. By leveraging the power of Next.js, Prisma, MySQL, and other technologies, it delivers a seamless e-commerce experience.

For more details, please visit the [live preview](https://ecommerce-store-cms.vercel.app/).
